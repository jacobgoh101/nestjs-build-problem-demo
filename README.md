## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ cd .. && yarn start backend database
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
